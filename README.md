## How To Use 

From your command line, clone and run react-portfolio-template:

```bash
# Clone this repository
$ git clone https://github.com/khayym/blog-react.git

# Go into the repository
$ cd blog-react

# Install dependencies
$ npm

#Start's development server
$ npm start
```
